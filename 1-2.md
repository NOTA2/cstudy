### 자료형 변환 (casting)
* 자동 형 변환 (묵시적 형 변환)
  * 대입연산 과정
    * 소수부의 손실 (double -> int)
    * 상위 바이트의 손실 (int -> char)
    * 정수의 승격 (int)
  * 피연산자의 자료형 불일치로 발생
    * int -> long -> long long -> float -> double -> long double
    * 데이터 손실의 최소화가 우선!
* 강제 형 변환 (명시적 형 변환)
  * 연산결과의 자료형은 피연산자의 자료형과 일치

### scanf 이해하기
`scanf("%d", &num)`

* 입력의 형식 : 어떻게 받아들일까
* 입력의 장소 : 어디에 저장할까

* 정수
  * %d
  * %o
  * %x
* 실수
  * float 형 : %f
  * double 형 : %lf
